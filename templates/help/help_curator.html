Curator preference

Most of the fields in the curator information setup are obvious, except for the following options.



  1. Highlight keywords. If you have input keywords in your project setup, all annotation pages will display matches by triggering Javascript. If this step takes too long, you can disable this flag.
  2. Show curated. After you have determined the relevance of each dataset, the default option is to not show it again when you open the annotation page again. But you can open this flag to review all determined datasets.
