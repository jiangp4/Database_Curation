Download curated data and automatic processing

Step 1: You can download all curated sample annotations and automatically processed expression matrices (if available). Please click CURATION -> RESULTS while you have selected a project.



Step 2: Then, go to the bottom page and click the Download button to download a file with URLs to sample annotations and data matrices.



Step 3: After getting the result file with URL list, you can download all files with "wget" or "python urllib", and develop an automatic analysis script. Here is an example python program to download files and process data from a download result file with URLs (from Step 2). In this example, curators annotated "Treatment, Condition, Sub Condtion, Curation, Dose" followed the Human Cytokine Response Project requirement. With these annotations and processed expression matrices from FDC, an automatic python script can generate differential expression profiles for treatments.

When you design the curation scheme, an essential criterion is to make sure that annotation fields should enable automatic analysis without further human intervention. If not, please redesign the annotation fields and curation schemes.
