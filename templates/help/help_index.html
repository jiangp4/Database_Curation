Curation Workflow

The curation procedure has three stages: identification of relevant datasets, annotation of sample information, and automatic data processing. Before any curation work, please get into main menu PROJECT -> SELECT to make the project selection.

  1. Stage One: Identify relevant datasets. Curators should identify datasets related to a specific study topic by reading the title, summary, design sections, and sample meta-information tables.
  2. Stage Two: Annotate sample conditions. For each identified relevant dataset, the curators should interpret the sample meta information and create standardized columns that automatic programs can analyze.
  3. Stage Three: Download and process data. Curators should design an automatic program to analyze data based on annotations from previous steps without further human interventions. (If not, please redesign the curation scheme.)

Note: Please use Google Chrome for the best performance. Firefox and Safari also work, although the regular expression functions will be compromised. This platform does NOT work on IE.

If you want a few examples, please see this Google Doc: Tutorial from the Human Cytokine Response project
