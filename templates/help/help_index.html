Curation Workflow

The curation procedure has two stages: identification of relevant datasets and the annotation of sample information. In phase one, curators should identify datasets related to a specific study topic by reading the title, summary, design sections, and sample meta-information tables. In step two, for each identified relevant dataset, the curators should interpret the sample meta information and create standardized columns that automatic programs can analyze. Before any curation work, please get into main menu PROJECT -> SELECT to make the project selection.
