SOFTWARE USE AGREEMENT FOR FRAMEWORK FOR DATA CURATION (FDC).

The Framework for Data Curation (FDC), proprietary to the National Cancer Institute (NCI), Cancer Data Science Laboratory, aims to help data scientists and biologists collect datasets deposited in the NCBI GEO and ArrayExpress databases. An FDC user (“Online Framework User”) can parse metadata information from deposited datasets and download preprocessed gene expression matrices.
